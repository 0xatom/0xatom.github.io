## Target IP

Always first step is to find target IP address, i prefer to use the `arp-scan` utility. Then we'll save the IP address in a variable. 

```
$ arp-scan --localnet | grep "VMware"
192.168.1.14	00:0c:29:11:aa:bc	VMware, Inc.
$ ip=192.168.1.14
```
